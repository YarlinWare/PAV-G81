Uso de las etiquetas Meta

Las etiquetas meta o meta tags encabezan un documento HTML y suministran información codificada a navegadores y motores de búsqueda sobre una página web. Los metadatos son invisibles para el usuario y se encargan de añadir información para facilitar el análisis de los archivos HTML y la gestión del contenido de una página web. Los meta tags de HTML siguen a menudo la misma construcción: en primer lugar se define un elemento y en segundo lugar se asigna un contenido.
